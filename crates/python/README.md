# NeMo Flow Python Bindings

Native Python bindings for NeMo Flow, built with PyO3 and packaged through Maturin.

The public Python package lives under `python/nemo_flow/`. Use the repository root `pyproject.toml` and `uv sync`
for local development and documentation builds.
