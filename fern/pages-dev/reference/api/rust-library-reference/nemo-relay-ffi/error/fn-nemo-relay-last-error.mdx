Retrieve the last error message set on this thread, or null if no error has occurred since the last [`clear_last_error`](/reference/api/rust-library-reference/nemo-relay-ffi/error/fn-clear-last-error) call.

The returned pointer borrows from thread-local storage and is valid only until the next FFI call on the same thread. Do **not** free the returned pointer.
