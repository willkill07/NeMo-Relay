Run the registered tool conditional execution guardrail chain.

Returns `NemoRelayStatus::Ok` if all guardrails pass, or `NemoRelayStatus::GuardrailRejected` if blocked.

## Parameters

- `name`: Tool name (null-terminated C string).
- `args_json`: Tool arguments as a JSON C string.

## Returns

Returns [`NemoRelayStatus::Ok`](/reference/api/rust-library-reference/nemo-relay-ffi/error/enum-nemorelaystatus) when execution is allowed and [`NemoRelayStatus::GuardrailRejected`](/reference/api/rust-library-reference/nemo-relay-ffi/error/enum-nemorelaystatus) when a guardrail blocks the call.

## Safety

All pointers must be valid.
