Machine-readable reason for the translation outcome.

Each variant describes WHY an intent received its status. This allows operators to distinguish between plugin limitations, backend limitations, policy decisions, and safety constraints.

## Variants

### `FullySupported`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "FullySupported"}} /></pre>

Intent was fully supported and applied.

### `UnsupportedByBackend`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "UnsupportedByBackend"}} /></pre>

Backend does not support this intent type at all.

### `UnsupportedByModel`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "UnsupportedByModel"}} /></pre>

Backend supports the intent but the specific model lacks the feature.

### `BackendLimitReached`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "BackendLimitReached"}} /></pre>

Intent was degraded due to backend-specific limits (e.g., max breakpoints).

### `InsufficientEvidence`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "InsufficientEvidence"}} /></pre>

Not enough historical evidence to apply the intent confidently.

### `FeatureDisabled`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "FeatureDisabled"}} /></pre>

The feature is available but administratively disabled.

### `UnsafeForRequest`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "UnsafeForRequest"}} /></pre>

Applying this intent would be unsafe for the current request.

### `PluginIncomplete`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "PluginIncomplete"}} /></pre>

The plugin implementation is incomplete for this intent type.

### `NotRelevant`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "NotRelevant"}} /></pre>

Intent was not relevant to the current request context.

### `Custom`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "Custom"}} /></pre>

Escape hatch for reason codes not yet in the enum.
