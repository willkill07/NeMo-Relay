Plugin registry for storing and retrieving provider plugins.

[`PluginRegistry`](/reference/api/rust-library-reference/nemo-relay-adaptive/acg/plugin_registry/struct-pluginregistry) is an instance-scoped (not global static) store for [`ProviderPlugin`](/reference/api/rust-library-reference/nemo-relay-adaptive/acg/plugin/trait-providerplugin) trait objects. Plugins are stored as `Arc<dyn ProviderPlugin>` keyed by plugin ID.

Instance-scoped design is more testable and can be promoted to a static later if needed.
