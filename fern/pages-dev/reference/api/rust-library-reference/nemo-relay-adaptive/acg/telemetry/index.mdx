Cache performance telemetry types for the Adaptive Cache Governor (ACG) system.

These types normalize provider-specific cache metrics (Anthropic `cache_read_input_tokens`/`cache_creation_input_tokens`, OpenAI `cached_tokens`) into a common schema for uniform measurement. Populated by provider-specific normalization logic in Phase 9.

## Structs

- [CacheHitRate](/reference/api/rust-library-reference/nemo-relay-adaptive/acg/telemetry/struct-cachehitrate): Aggregated cache hit rate over a time window.
- [CacheMissDiagnosis](/reference/api/rust-library-reference/nemo-relay-adaptive/acg/telemetry/struct-cachemissdiagnosis): Structured diagnosis for a cache miss.
- [CacheRequestFacts](/reference/api/rust-library-reference/nemo-relay-adaptive/acg/telemetry/struct-cacherequestfacts): Request-time facts used to classify a cache miss without leaking prompt text.
- [CacheTelemetryEvent](/reference/api/rust-library-reference/nemo-relay-adaptive/acg/telemetry/struct-cachetelemetryevent): Per-call cache telemetry event.
