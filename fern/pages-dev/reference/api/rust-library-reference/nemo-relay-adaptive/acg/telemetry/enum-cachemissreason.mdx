Reason why a cache miss occurred.

Covers 8 determinable reasons plus an extensible `Other` variant. Uses internally-tagged JSON representation (`"reason"` field) so each variant serializes as `{"reason": "snake_case"}` and the `Other` variant additionally carries a `description` field.

## Variants

### `PrefixMismatch`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "PrefixMismatch"}} /></pre>

Prompt prefix didn't match cached prefix.

### `BelowMinimumThreshold`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "BelowMinimumThreshold"}} /></pre>

Stable prefix shorter than provider minimum for caching.

### `RetentionExpired`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "RetentionExpired"}} /></pre>

Cached prefix retention window elapsed.

### `RoutingMismatch`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "RoutingMismatch"}} /></pre>

Request routed to different worker/pool.

### `Evicted`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "Evicted"}} /></pre>

Cache evicted due to capacity pressure.

### `UnsupportedFeature`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "UnsupportedFeature"}} /></pre>

Backend/model doesn't support caching.

### `ColdStart`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "ColdStart"}} /></pre>

First request for this prefix (no prior cache entry).

### `Unknown`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "Unknown"}} /></pre>

Reason could not be determined from provider response.

### `Other`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "Other"}} /></pre>

Extensible escape hatch for reasons not yet in the enum.
