Return the current scope at the top of the active stack.

This reads the task-local or thread-local scope stack without mutating it and returns a clone of the current top-most [`ScopeHandle`](/reference/api/rust-library-reference/nemo-relay/api/scope/struct-scopehandle).

## Returns

A [`Result`](/reference/api/rust-library-reference/nemo-relay/error/type-result) containing the current [`ScopeHandle`](/reference/api/rust-library-reference/nemo-relay/api/scope/struct-scopehandle) when the runtime owner check succeeds.

## Errors

Returns an error when the current binding has not initialized the shared runtime ownership correctly.
