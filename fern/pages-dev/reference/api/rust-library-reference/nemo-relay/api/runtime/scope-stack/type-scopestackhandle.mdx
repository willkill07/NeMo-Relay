Shared handle type for the runtime scope stack.

The runtime stores the active [`ScopeStack`](/reference/api/rust-library-reference/nemo-relay/api/runtime/scope_stack/struct-scopestack) behind an [`Arc`](https://doc.rust-lang.org/1.93.0/alloc/sync/struct.Arc.html) and [`RwLock`](https://doc.rust-lang.org/1.93.0/std/sync/poison/rwlock/struct.RwLock.html) so bindings can propagate it across execution contexts while still allowing concurrent readers.
