Capture the current thread-local scope stack binding.

This is intended for foreign runtimes that temporarily bind a scope stack to an OS thread and need to restore the exact previous state before releasing that thread back to their scheduler.

## Returns

A [`ThreadScopeStackBinding`](/reference/api/rust-library-reference/nemo-relay/api/runtime/scope_stack/struct-threadscopestackbinding) containing the current thread-local stack and explicit-binding flag.
