Continuation type invoked by tool execution intercepts.

Execution intercepts receive this callable as their `next` continuation and can call it with modified arguments, wrap it, or skip it entirely.

## Parameters

- First argument: JSON argument payload to pass to the remaining execution chain.

## Returns

A future resolving to the tool result JSON.

## Errors

The future resolves to an error when the remaining execution chain fails.
