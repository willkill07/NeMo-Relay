Create a new scope stack handle with an implicit root scope.

The returned handle wraps a freshly initialized [`ScopeStack`](/reference/api/rust-library-reference/nemo-relay/api/runtime/scope_stack/struct-scopestack) inside an [`Arc`](https://doc.rust-lang.org/1.93.0/alloc/sync/struct.Arc.html) and [`RwLock`](https://doc.rust-lang.org/1.93.0/std/sync/poison/rwlock/struct.RwLock.html) so it can be shared across async tasks or threads.

## Returns

A new [`ScopeStackHandle`](/reference/api/rust-library-reference/nemo-relay/api/runtime/scope_stack/type-scopestackhandle) containing exactly one implicit root scope.

## Notes

The root scope is always present and cannot be removed.
