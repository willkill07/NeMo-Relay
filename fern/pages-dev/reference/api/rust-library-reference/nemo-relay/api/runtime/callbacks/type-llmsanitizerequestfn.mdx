Sanitize an LLM request before the runtime records it.

LLM request sanitizers affect the serialized request payload emitted on start events. They do not mutate the caller-owned [`LlmRequest`](/reference/api/rust-library-reference/nemo-relay/api/llm/struct-llmrequest) unless a separate request intercept does so.

## Parameters

- First argument: LLM request payload to sanitize for observability.

## Returns

Sanitized [`LlmRequest`](/reference/api/rust-library-reference/nemo-relay/api/llm/struct-llmrequest) for the emitted event.
