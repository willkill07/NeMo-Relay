Register a global LLM request intercept. Request intercepts can rewrite or annotate the outgoing LLM request.

## Parameters

- `name`: Unique middleware name in the global registry.
- `priority`: Lower values run earlier in the chain.
- `break_chain`: Whether the intercept should stop later request intercepts after it returns.
- `callable`: Intercept callback stored under `name`.

## Returns

A [`Result`](/reference/api/rust-library-reference/nemo-relay/error/type-result) that is `Ok(())` when the intercept was registered.

## Errors

Returns [`FlowError::AlreadyExists`](/reference/api/rust-library-reference/nemo-relay/error/enum-flowerror) when the name is already in use or an internal error if the runtime state cannot be updated.
