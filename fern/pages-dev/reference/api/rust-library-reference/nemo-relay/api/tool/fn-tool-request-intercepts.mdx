Run only the tool request-intercept chain.

This applies the currently active global and scope-local request intercepts without emitting lifecycle events or invoking tool execution.

## Parameters

- `name`: Tool name used when resolving the intercept chain.
- `args`: Raw tool arguments to transform.

## Returns

A [`Result`](/reference/api/rust-library-reference/nemo-relay/error/type-result) containing the transformed JSON arguments.

## Errors

Returns any error raised by the request-intercept chain.

## Notes

Conditional guardrails and execution intercepts are not run by this helper.
