Pluggable pricing source interface.

Database, service-backed, or enterprise-managed pricing integrations should implement this trait and return a validated catalog snapshot. The LLM hot path uses [`PricingResolver`](/reference/api/rust-library-reference/nemo-relay/codec/pricing/struct-pricingresolver), so sources can refresh out-of-band without making each response decode perform network or database I/O.
