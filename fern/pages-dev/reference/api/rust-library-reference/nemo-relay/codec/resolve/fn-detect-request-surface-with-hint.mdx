Like [`detect_request_surface`](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-detect-request-surface), but a recognized `provider_hint` resolves the one ambiguous shape (an Anthropic request without a top-level `system`, otherwise read as OpenAI Chat). Today, `"anthropic"` is the only hint that changes detection; `None` or any other value is ignored and detection stays shape-only.
