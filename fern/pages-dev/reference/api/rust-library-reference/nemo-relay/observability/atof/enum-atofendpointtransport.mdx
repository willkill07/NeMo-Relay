## Variants

### `HttpPost`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "HttpPost"}} /></pre>

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "Ndjson"}} /></pre>
